# 知识库总览（knowledge）

本目录下是整个知识库的根目录，按不同业务/主题划分为多个领域子目录。

- Safety Knowledge/
  - 用途：安全相关知识与文档，涵盖常见 Web 安全漏洞及防护方案。
- Financial Report Data/
  - 用途：金融与上市公司财报类数据，主要是按公司和季度划分的 PDF 报告。
- E-commerce Data/
  - 用途：电商业务相关的数据文件，如员工、客户、库存和订单等结构化数据,以及电商中用户与客服间常见问题问答数据。
- AI Knowledge/
  - 用途：人工智能行业与技术相关报告与研究资料，用于检索 AI 技术发展、应用场景及治理趋势。
- multihop-news/
  - 用途：MultiHop-RAG 新闻数据集，用于多跳问答评估，包含跨文档推理的新闻文章。
- legal/
  - 用途：法律知识库，包含成文法、合同范本、判例。支持 CUAD 41 类合同条款检索。
  - 子目录：statutes/（成文法）、contracts/（合同范本）、cases/（判例）
- medical/
  - 用途：医学知识库，遵循循证医学 PICO 框架和证据金字塔。
  - 子目录：guidelines/（临床指南）、drugs/（药品说明书）、diseases/（疾病诊疗）、literature/（循证文献）
